# Contributing Guide

Thank you for wanting to improve this repo! Here's how:

## Adding a New Notebook
1. Follow the notebook template in README.md
2. Add the difficulty tag: `🟢 Beginner | 🟡 Intermediate | 🔴 Advanced`
3. Include working code examples with expected outputs as comments
4. Add a "Why AI Engineers need this" section

## Reporting Issues
Open an issue describing: the notebook name, what's wrong, and the correct version.

## Style Guide
- Use `f-strings` over `.format()` or `%`
- Include type hints in function signatures
- Explain the *why*, not just the *what*
