## 🚀 How to Use This Repo

**Option 1 — Google Colab (recommended, no setup):**
Click the `Open in Colab` badge at the top of any notebook.

**Option 2 — Local Setup:**
```bash
git clone https://github.com/yourusername/python-for-ai-engineers.git
cd python-for-ai-engineers
pip install -r requirements.txt
jupyter notebook
```

## 📖 Notebook Structure
Every notebook follows this consistent template:

```
📌 What is it?         — Plain English definition
🧠 Why AI Engineers need it — Real-world context
✏️  Syntax             — Minimal, clean syntax reference
💡 Examples            — 3–5 progressive examples
⚠️  Common Mistakes    — Gotchas and pitfalls
🔗 What's Next?        — Link to the next concept
```
